# NOVA

> **Notation for Open and Versatile Archives**  
> **Record anything. Define everything.**

NOVAは、記録対象の構造とルールを自分で定義し、あらゆる情報を残せる汎用アーカイブサービスです。構造化されたProtocol Recordと、すぐに書き残せるOpen Recordを共通の履歴モデルで扱います。

現在は設計・初期開発段階です。
